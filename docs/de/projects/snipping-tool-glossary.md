# Glossar – Snipping Tool

Dieses Glossar definiert verbindliche UI-Bezeichnungen für die Dokumentation
des Windows Snipping Tools.  
Die Begriffe entsprechen den Beschriftungen in der Benutzeroberfläche.

---

| Deutscher Begriff | Englischer UI-Name | Beschreibung |
|------------------|-------------------|--------------|
| Ausschnitt | Snipping area | Zentraler Bereich zur Auswahl von Screenshot- und Video-Funktionen |
| Ausschnittbereich | Snipping area | Der tatsächlich ausgewählte Bildschirmbereich |
| Ausschnitt nach Verzögerung | Delay snip | Verzögertes Erstellen eines Screenshots |
| Textextraktor | Text extractor | Funktion zum Erkennen und Kopieren von Text aus Bildern (OCR) |
| Farbwähler | Color picker | Werkzeug zum Erfassen eines Farbwerts vom Bildschirm |
| Bildschirmfoto | Screenshot | Einzelbildaufnahme des Bildschirms |
| Bildschirmvideo | Screen recording | Videoaufnahme eines Bildschirmbereichs |
| Farbwert | Color value | Ergebnis der Farbauswahl (z. B. Hex- oder RGB-Wert) |

---

**Hinweis:**  
UI-Bezeichnungen werden in der Dokumentation exakt so verwendet,
wie sie in der jeweiligen App-Sprache angezeigt werden.
