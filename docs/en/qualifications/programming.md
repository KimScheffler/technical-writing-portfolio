# Professional Developement

## 100 Days of Code™ – The Complete Python Pro Bootcamp
*Ongoing professional development*

Practice-oriented online course focused on building a solid foundation in Python
programming through hands-on projects.

The course covers Python fundamentals as well as practical applications in
automation, data processing, and basic web development, with an emphasis on
writing clean, readable, and maintainable code.

Key topics include:
- Python 3 fundamentals and scripting
- automation and small tooling projects
- data handling and basic data analysis
- working with APIs and web data
- version control using Git and GitHub
- introductory web development concepts

The course follows a project-based learning approach, with incremental exercises
and practical assignments to reinforce concepts and support real-world application.

The course supports my technical writing work by strengthening my ability to
understand, evaluate, and automate technical workflows, as well as to document
code-related processes and tools more effectively.
